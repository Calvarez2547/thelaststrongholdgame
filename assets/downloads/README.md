# Download Placeholder

Place your final Windows beta ZIP file here if you want Cloudflare Pages to host it directly.

Suggested filename:

- `the-last-stronghold-beta.zip`

Then set `GAME_DOWNLOAD_URL` in Cloudflare Pages to the full public URL, for example:

`https://thelaststrongholdgame.pages.dev/assets/downloads/the-last-stronghold-beta.zip`

If you use a different ZIP filename, update only `GAME_DOWNLOAD_URL`. The frontend download flow can stay the same.
